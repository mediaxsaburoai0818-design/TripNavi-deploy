          <p>アメリカへの短期滞在にはESTA（エスタ）申請が必須ですが、2023年より米国CBPが公式アプリ<strong>「ESTA Mobile」</strong>の提供を開始しました。この記事では、スマホアプリでのESTA申請方法と、公式サイトとの違いを詳しく解説します。</p>

          
          <h2 id="section1">「ESTA Mobile」について</h2>
          <p><strong>ESTA Mobile</strong>は、米国税関・国境警備局（CBP）が2023年より提供している公式ESTAアプリです。スマホ1台で、<strong>申請から支払い、ステータス確認まで完結</strong>できます。</p>

          <div class="info-box">
            <h4>📱 ESTA Mobileの特徴</h4>
            <ul>
              <li><strong>iOS/Android対応</strong></li>
              <li><strong>パスポート自動読み取り</strong>（OCR機能でスペルミス防止）</li>
              <li><strong>ICチップスキャン</strong>で情報自動入力</li>
              <li><strong>カメラで顔写真撮影</strong>→そのままアップロード</li>
              <li><strong>申請ステータスをアプリで確認</strong>可能</li>
            </ul>
          </div>

          <p><strong>ダウンロード:</strong></p>
          <ul>
            <li><strong>iOS:</strong> App Store</li>
            <li><strong>Android:</strong> Google Play</li>
          </ul>

          <h2 id="section2">ESTA Mobileでの申請で必要なもの</h2>
          
          <div class="info-box">
            <h4>📋 必要なもの</h4>
            <ul>
              <li><strong>有効なパスポート</strong>（ICチップ搭載、有効期限が渡航終了日から6ヶ月以上）</li>
              <li><strong>メールアドレス</strong>（申請完了通知受信用）</li>
              <li><strong>クレジットカード/デビットカード/PayPal</strong>（支払い用）</li>
              <li><strong>パスポートの顔写真ページ</strong>（カメラで撮影）</li>
              <li><strong>申請者本人の顔写真</strong>（アプリ内で新規撮影、無地背景、上半身が鮮明に写っていること）</li>
            </ul>
          </div>

          
          <h2 id="section3">ESTA Mobileでの申請方法</h2>
          <p>ESTA Mobileでの申請は、以下の5ステップで完了します。</p>

          <h3 id="section3-1">1. モバイルアプリ「ESTA Mobile」をインストール</h3>
          <ol>
            <li>App StoreまたはGoogle Playから<strong>「ESTA Mobile」</strong>をダウンロード</li>
            <li>アプリを起動し、<strong>「操作の継続」</strong>をタップ</li>
            <li><strong>「セキュリティ通告」</strong>を確認し、<strong>「同意」</strong>をタップ</li>
            <li>ホーム画面で<strong>「開始する」</strong>をタップ</li>
            <li>免責事項を確認し、<strong>「同意」</strong>→<strong>「操作の継続」</strong></li>
          </ol>

          <h3 id="section3-2">2. パスポートの顔写真をスキャン</h3>
          <ol>
            <li><strong>「スキャンの開始」</strong>をタップ</li>
            <li>カメラアクセスを<strong>「許可」</strong></li>
            <li>白い枠内にパスポートの顔写真ページを収める</li>
            <li>自動読み取り完了後、<strong>「正常にスキャンができました」</strong>と表示される</li>
            <li>パスポート番号、生年月日などが自動反映されるので確認</li>
          </ol>

          <div class="warn-box">
            <h4>⚠️ 注意</h4>
            <p>光の反射や影でうまく読み取れない場合があります。明るい場所で撮影してください。</p>
          </div>

          <h3 id="section3-3">3. パスポートのICチップの読み取り</h3>
          <ol>
            <li>パスポートの<strong>裏表紙側</strong>の上にスマホを重ねる</li>
            <li>エラーが出る場合は、スマホケースを外し、パスポートの裏表紙側に当てる</li>
            <li><strong>「スキャンの準備ができました」</strong>と表示される</li>
            <li>氏名、性別、生年月日、有効期限が表示されるので確認</li>
            <li><strong>「操作の継続」</strong>をタップ</li>
          </ol>

          <h3 id="section3-4">4. 顔写真の撮影と個人情報の入力</h3>
          <ol>
            <li>青い枠内に顔を収めて撮影（無地背景、上半身が鮮明に写るように）</li>
            <li>生年月日、パスポート番号などが表示されるので確認</li>
            <li><strong>「操作の継続」</strong>をタップ</li>
            <li>住所、電話番号、勤務先、アメリカでの滞在先を<strong>英語（ローマ字）</strong>で入力</li>
            <li>伝染病、犯罪歴などの質問に<strong>「はい/いいえ」</strong>で回答</li>
          </ol>

          <h3 id="section3-5">5. 申請料金の支払い</h3>
          <ol>
            <li>最終確認を行い、支払いページへ進む</li>
            <li>料金: <strong>40.27米ドル/人</strong></li>
            <li>支払い期限: 申請日から<strong>6日以内</strong></li>
            <li>支払い方法: Visa/Mastercard/JCB/AMEX/Diners/PayPal</li>
            <li><strong>「PAY NOW」</strong>をタップして決済完了</li>
            <li>申請番号が表示されるので<strong>スクリーンショットで保存</strong>（ステータス確認時に必要）</li>
          </ol>

          
          <h2 id="section4">申請時の注意点</h2>
          
          <div class="warn-box">
            <h4>⚠️ 重要な注意点</h4>
            
            <p><strong>申請期限:</strong></p>
            <p>出発<strong>72時間前（3日前）まで</strong>に申請を完了することを推奨します。審査には最大3日かかる場合があります。</p>

            <p><strong>ステータス確認:</strong></p>
            <ul>
              <li>アプリのホーム画面で<strong>「見つける」</strong>をタップ</li>
              <li>氏名、生年月日、申請番号を入力</li>
              <li>審査中: <strong>「保留中」</strong>と表示</li>
              <li>結果: <strong>「渡航認証許可」</strong>または<strong>「渡航認証拒否」</strong></li>
              <li>⚠️ 画面を横スワイプすると無効化される可能性があるため注意</li>
            </ul>

            <p><strong>料金改定:</strong></p>
            <ul>
              <li>2025年9月29日まで: 21米ドル</li>
              <li>2025年9月30日〜: 40米ドル（大幅改定）</li>
              <li>2026年1月1日〜: <strong>40.27米ドル</strong>（インフレ調整）</li>
              <li>今後も年次調整が予定されています</li>
            </ul>
          </div>

          
          <h2 id="section5">よくある質問</h2>

          <div class="faq-item">
            <h3>Q1: ESTAの申請はスマホでできますか？</h3>
            <p><strong>A:</strong> はい、可能です。公式アプリ<strong>「ESTA Mobile」</strong>をインストールすることで、スマホから申請できます。カメラでパスポートを撮影し、ICチップをスキャンする必要があります。公式サイトからの申請とは手順が一部異なります。</p>
          </div>

          <div class="faq-item">
            <h3>Q2: ESTAの入力は日本語にならない？</h3>
            <p><strong>A:</strong> ESTA Mobileアプリは日本語表示に切り替え可能です。ただし、<strong>入力は英語（ローマ字）のみ</strong>です。スペルミスがあると再申請が必要になり、虚偽申請と見なされると入国拒否の恐れがあるため、慎重に入力してください。</p>
          </div>

          <div class="faq-item">
            <h3>Q3: ESTAの料金は2025年にいくらになりますか？</h3>
            <p><strong>A:</strong> ESTA申請料金は以下の通りです：</p>
            <ul>
              <li><strong>2025年9月29日まで:</strong> 21米ドル</li>
              <li><strong>2025年9月30日〜:</strong> 40米ドル（大幅改定）</li>
              <li><strong>2026年1月1日〜:</strong> 40.27米ドル（インフレ調整）</li>
            </ul>
            <p>今後も年次調整が予定されています。</p>
          </div>

          
          <h2 id="section6">まとめ</h2>
          <p>アメリカへの短期滞在にはESTA申請が必須です。2023年より公式アプリ<strong>「ESTA Mobile」</strong>が提供開始され、スマホで申請から支払い、ステータス確認まで完結できるようになりました。</p>

          <p>パスポート撮影やICチップ読み取りなど、公式サイトとは手順が一部異なるため、事前に手順を確認してから申請を進めることを推奨します。</p>

          <div class="info-box">
            <h4>✅ ESTA Mobileのポイント</h4>
            <ul>
              <li>スマホ1台で申請完結</li>
              <li>パスポート自動読み取りでスペルミス防止</li>
              <li>ICチップスキャンで情報自動入力</li>
              <li>申請料金: 40.27米ドル（2026年1月1日〜）</li>
              <li>出発72時間前までに申請完了推奨</li>
              <li>iOS/Android対応</li>
