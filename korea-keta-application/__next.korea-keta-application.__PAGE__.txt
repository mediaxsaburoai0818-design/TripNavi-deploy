          <p>韓国への旅行を計画している方の中には、「K-ETAは必要なの？」と疑問に思う方も多いでしょう。この記事では、<strong>K-ETA（電子旅行許可証）</strong>の概要と、現在実施されている一時免除措置について詳しく解説します。</p>

          
          <h2 id="section1">K-ETAとは</h2>
          <p><strong>K-ETA（Korea Electronic Travel Authorization）</strong>は、2021年9月に導入された電子旅行認証制度です。ビザ免除国の国民が90日以内の観光やビジネス目的で韓国に入国する際に必要となります。アメリカのESTAと同様の仕組みです。</p>

          <div class="info-box">
            <h4>📌 K-ETAの特徴</h4>
            <ul>
              <li><strong>導入日:</strong> 2021年9月</li>
              <li><strong>有効期限:</strong> 承認日から3年間（2023年7月2日以降）</li>
              <li><strong>申請料金:</strong> ₩10,300（決済手数料含む）</li>
              <li><strong>対象:</strong> 90日以内の観光・ビジネス</li>
              <li><strong>複数回渡航:</strong> 有効期限内なら何度でも入国可能（1回の滞在は最長90日）</li>
            </ul>
          </div>

          <h3 id="section1-1">2026年12月までK-ETA一時免除措置が延長</h3>
          
          <div class="info-box">
            <h4>✨ 重要：一時免除措置（2026年12月31日まで）</h4>
            <p>日本を含む22の国・地域が対象となる一時免除措置が実施されています：</p>
            <ul>
              <li><strong>免除期間:</strong> 2026年12月31日まで延長</li>
              <li><strong>対象:</strong> 日本を含む22か国・地域</li>
              <li><strong>免除内容:</strong> 免除期間中はK-ETA不要で韓国入国可能</li>
              <li><strong>滞在目的:</strong> 観光、商用、親族訪問（90日以内）</li>
            </ul>
          </div>

          <p><strong>対象国・地域:</strong></p>
          <p>日本、オランダ、ノルウェー、ニュージーランド、台湾、デンマーク、ドイツ、マカオ、アメリカ、ベルギー、スウェーデン、スペイン、シンガポール、イギリス、オーストリア、イタリア、カナダ、ポーランド、フランス、フィンランド、オーストラリア、香港</p>

          <h3 id="section1-2">K-ETAの申請対象者</h3>
          <p><strong>K-ETA申請が必要な方:</strong></p>
          <ul>
            <li>ビザ免除国で90日以内の短期滞在をする方</li>
            <li>トランジットで空港外に出る方</li>
          </ul>

          <p><strong>K-ETA申請が不要な方:</strong></p>
          <ul>
            <li>韓国ビザを取得済みの方</li>
            <li>一時免除措置期間中の対象国民</li>
            <li>トランジットのみ（空港外に出ない）の方</li>
          </ul>

          <h3 id="section1-3">K-ETAの申請料金</h3>
          <p>申請料金は<strong>₩10,300</strong>（決済手数料含む）です。</p>

          <h3 id="section1-4">K-ETAの有効期限</h3>
          <p>承認日から<strong>3年間</strong>有効です（2023年7月2日以前は2年間でした）。ただし、パスポートの有効期限が3年未満の場合は、パスポート有効期限日までとなります。</p>

          <h2 id="section3">K-ETA申請に必要な書類</h2>
          
          <div class="info-box">
            <h4>📋 必要な書類</h4>
            <ul>
              <li><strong>パスポート</strong>（ICチップ搭載、有効期限内、残存3か月以上）</li>
              <li><strong>顔写真データ</strong>（JPEG、6か月以内撮影、白背景、100KB以下、700px以下）</li>
              <li><strong>クレジットカード/デビットカード</strong>（支払い用）</li>
              <li><strong>メールアドレス</strong>（認証用）</li>
            </ul>
          </div>

          <div class="info-box">
            <h4>📝 必要な情報</h4>
            <ul>
              <li>個人情報、パスポート情報、連絡先情報</li>
              <li>韓国での滞在先情報</li>
              <li>職業、緊急連絡先</li>
              <li>入国目的</li>
              <li>感染症・犯罪歴の有無</li>
            </ul>
          </div>

          
          <h2 id="section4">K-ETAの申請方法</h2>
          <p>K-ETAの申請は、以下の10ステップで完了します。</p>

          <h3>1. 公式ウェブサイトの申請ページへアクセス</h3>
          <p>K-ETA公式サイト（<strong>https://www.k-eta.go.kr/</strong>）にアクセスし、日本語を選択、「<strong>K-ETAを申請する</strong>」をクリックします。</p>

          <h3>2. 利用規約の確認、メール認証</h3>
          <p>利用規約に同意し、メールアドレスを登録。6桁の認証番号を入力します。</p>

          <h3>3. パスポート情報の入力</h3>
          <p>パスポート画像をアップロードするか、手入力でパスポート情報を登録します。</p>

          <h3>4. 連絡先情報の入力</h3>
          <p>電話番号、メールアドレス、住所を入力します。</p>

          <h3>5. 渡航情報・滞在先情報の入力</h3>
          <p>入国予定日、滞在期間、宿泊先を入力します。</p>

          <h3>6. 職業と緊急連絡先の入力</h3>
          <p>職業カテゴリを選択し、緊急連絡先を入力。感染症・犯罪歴に関する質問に回答します。</p>

          <h3>7. 顔写真のアップロード</h3>
          <p>JPEG形式、100KB以下、700px以下の顔写真をアップロードします。</p>

          <h3>8. 入力内容の確認、申請料金の決済</h3>
          <p>入力内容を最終確認し、<strong>₩10,300</strong>を決済します。</p>

          <h3>9. K-ETA申請の完了</h3>
          <p>受付完了メールを受領し、申請番号を保管します。</p>

          <h3>10. 申請状況・審査結果の確認</h3>
          <p>通常、数時間〜72時間以内に審査結果が通知されます。<strong>Approved（承認）</strong>、<strong>Selectee（追加審査）</strong>、<strong>Not OK（不承認）</strong>のいずれかが表示されます。</p>

          
          <h2 id="section5">よくある質問</h2>

          <div class="faq-item">
            <h3>Q1: 日本人でもK-ETAは必要ですか？</h3>
            <p><strong>A:</strong> <strong>2026年12月31日まで</strong>は一時免除措置により、K-ETA不要です。条件は以下の通りです：</p>
            <ul>
              <li>有効なパスポート保持</li>
              <li>90日以内の短期滞在</li>
              <li>e-Arrival Card提出</li>
            </ul>
            <p>免除措置終了後は、K-ETA申請が必要になります。</p>
          </div>

          <div class="faq-item">
            <h3>Q2: K-ETAなしで韓国に入国できますか？</h3>
            <p><strong>A:</strong> 以下の場合は可能です：</p>
            <ul>
              <li>一時免除措置対象国で、免除期間中の入国</li>
              <li>有効な韓国ビザを取得済み</li>
            </ul>
          </div>

          <div class="faq-item">
            <h3>Q3: K-ETAの免除は2026年までですか？</h3>
            <p><strong>A:</strong> 現時点では<strong>2026年12月31日まで</strong>です。当初2024年12月31日まででしたが、延長されました。今後も延長または短縮される可能性があります。</p>
          </div>

          <div class="faq-item">
            <h3>Q4: 韓国に入国するにはK-ETAが必要ですか？</h3>
            <p><strong>A:</strong> 国籍、渡航時期、目的により異なります：</p>
            <ul>
              <li><strong>通常:</strong> 日本人がビザなしで90日以内の観光・商用 → K-ETA必要</li>
              <li><strong>現在:</strong> 一時免除措置期間中 → K-ETA不要</li>
            </ul>
          </div>

          <div class="faq-item">
            <h3>Q5: K-ETAはいつまで有効ですか？</h3>
            <p><strong>A:</strong> 承認日から<strong>3年間</strong>有効です。ただし、パスポート有効期限が3年未満の場合は、その期限までとなります。有効期限内は複数回渡航可能で、1回の滞在は最長90日です。</p>
          </div>

          <div class="faq-item">
            <h3>Q6: K-ETAは申請後に内容を変更できますか？</h3>
            <p><strong>A:</strong> 審査開始後は変更不可です。入力ミスも修正できません。パスポート更新や氏名変更時は、再申請が必要です。</p>
          </div>

          
          <h2 id="section6">まとめ</h2>
          <p><strong>K-ETA</strong>は韓国の電子旅行許可制度です。現在、<strong>2026年12月31日まで</strong>、日本人は一時免除措置によりK-ETA不要で韓国に入国できます。</p>

          <p>免除措置終了後は、K-ETA申請が必要となります。料金は<strong>₩10,300</strong>、有効期限は<strong>3年間</strong>です。公式ウェブサイトから10ステップで申請できます。</p>

          <div class="info-box">
            <h4>✅ K-ETA申請のポイント</h4>
            <ul>
              <li>2026年12月31日まで日本人は一時免除措置でK-ETA不要</li>
              <li>申請料金: ₩10,300</li>
              <li>有効期限: 3年間（複数回渡航可能）</li>
              <li>申請方法: 公式ウェブサイトから10ステップ</li>
              <li>審査時間: 数時間〜72時間</li>
              <li>免除措置終了後は申請必要</li>
            </ul>
          </div>
